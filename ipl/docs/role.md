# Role

**Admin RT:** 
- Membuat data iuran
- Membuat data warga
- Membuat data iuran
- enroll iuran ke warga

**Bendahara:** 
- mencatat transaksi pembayaran iuran, 
- melihat laporan daftar iuran
- melihat laporan transaksi pembayaran iuran

**Ketua RT:** 
- Melihat data warga
- Melihat data iuran 
- Melihat laporan transaksi iuran

**Warga:** 
- Melihat data warga pribadi
- Melihat data iuran pribadi
- Melihat data transaksi iuran pribadi
