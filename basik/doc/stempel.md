# Stempel

Pada bagian ini kita akan belajar bagaimana menstempel di kanvas. Kita akan menggunakan perintah `stempel()`.

```
stempel("kotak");
```

Jalankan perintah ini maka akan terlihat gambar kotak di layar.

`stempel()` akan menstempel gambar yang di muat dari file. Semua file yang bisa di stempel di simpan di folder `asset`.
Kita bisa menambahkan file ke folder ini

Perintah `stempel("kotak")` artinya kita menempelkan gambar yang di muat dari file 'kotak.png' ke layar.

Latihan:

- Coba ganti dengan file yang lain.
- Coba masukkan gambar yang kamu punya ke folder asset dan gunakan perintah `stempel()`

## Menaruh pada posisi tertentu

Perintah `stempel()` bisa menerima parameter tambahan untuk menentukan posisi gambar. Contoh:

```
stempel("kotak", 100, 100);
```

Akan menstempel gambar kotak ke poisi 100,100

Latihan:

- Cobalah untuk menstempel beberapa gambar. Misal: gambar pemandangan, pohon, rumah, dsb.
