# Berlatih menggambar rumah

Sekarang mari kita belajar menggambar rumah:

Pertama kita akan menggambar atapnya:

```
garis(200, 50, 300, 150);
garis(200, 50, 100, 150);
garis(100, 150, 300, 150);
```

Jalankan kodenya dengan menekan tombol "Jalankan", hasilnya akan seperti ini:
[gambar atap]

Kita lanjutkan untuk menggambar bagian bawah, tambahkan baris berikut:

```
garis(100, 150, 100, 250);
garis(100, 250, 300, 250);
garis(300, 250, 300, 150);
```

Jalankan lagi dan sekarang hasilnya akan seperti ini:
[gambar rumah]

Kode secara lengkap adalah sebagai berikut:

```
garis(200, 50, 300, 150);
garis(200, 50, 100, 150);
garis(100, 150, 300, 150);

garis(100, 150, 100, 250);
garis(100, 250, 300, 250);
garis(300, 250, 300, 150);
```

## Latihan

Coba tambahkan pintu dan jendela. Pintu bisa berbentuk persegi panjang dengan jendela berbentuk kotak disampingnya.
