Berikut adalah **fitur minimal (Minimum Viable Product / MVP)** yang perlu dimiliki sebuah game **platformer 2D** bergaya *Super Mario Bros* klasik. Daftar ini fokus pada elemen paling esensial supaya game sudah bisa dimainkan, diuji, dan dikembangkan lebih lanjut.

***

# 🎮 Fitur Minimal untuk MVP Game ala *Mario Bros Classic*

## 1. **Core Gameplay Mechanics**

Ini adalah fondasi agar game bisa dimainkan.

### ✔️ **Gerakan Karakter Utama**

*   Bergerak kiri/kanan
*   Melompat (dengan gravitasi dan collision yang konsisten)
*   Aksi dasar seperti lari (opsional untuk MVP, tapi lebih baik ada)

### ✔️ **Collision System**

*   Interaksi dengan lantai, dinding, dan platform
*   Karakter tidak jatuh menembus map
*   Karakter bisa berdiri di atas platform bergerak (opsional tapi bagus)

### ✔️ **Kamera yang mengikuti karakter**

*   Side-scrolling horizontal
*   Tidak perlu super halus, cukup fungsional

***

## 2. **Level dan Lingkungan Dasar**

### ✔️ **1 Level yang Playable**

*   Platform, celah, area loncat
*   Pintu/flag/titik akhir level
*   Tata letak sederhana untuk playtest

### ✔️ **Tileset / Obstacles Sederhana**

*   Blok lantai
*   Blok solid
*   Lubang
*   (Opsional) Platform bergerak

***

## 3. **Enemy dan Interaksi Dasar**

### ✔️ **Minimal 1 tipe musuh**

Contoh: Goomba-like

*   Bisa bergerak lurus
*   Mati saat diinjak

### ✔️ **Player mengambil damage / mati**

*   Kalah bila kena musuh atau jatuh ke jurang
*   Restart level / checkpoint sederhana

***

## 4. **Audio Minimal**

*   SFX lompat
*   SFX mati
*   SFX menginjak musuh
*   (Opsional) BGM sederhana

Tidak harus kompleks—cukup fungsional untuk feedback pemain.

***

## 5. **UI Fungsional**

*   Start screen sederhana (Play)
*   HUD minimal: jumlah nyawa atau skor
*   Game Over screen

***

## 6. **Basic Game Loop**

Siklus game yang sudah bekerja:

1.  Player mulai game
2.  Player bermain di 1 level
3.  Menang (finish) atau kalah
4.  Kembali ke menu atau restart

***

# ✨ Bonus (Tidak wajib untuk MVP tetapi sangat membantu)

*   Sistem power-up (membesar, fireball, dsb.)
*   Beberapa level kecil
*   Animasi karakter lebih lengkap (run, jump, idle)

***

# 🧩 Ringkasan Singkat (Checklist)

| Fitur                   | Wajib untuk MVP |
| ----------------------- | --------------- |
| Gerak + Lompat          | ✔️              |
| Collision system        | ✔️              |
| 1 level lengkap         | ✔️              |
| Minimal 1 musuh         | ✔️              |
| Player mati / win       | ✔️              |
| UI dasar                | ✔️              |
| Audio dasar             | ✔️              |
| Kamera mengikuti player | ✔️              |
